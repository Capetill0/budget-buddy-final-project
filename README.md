# budget-buddy-final-project
A personal finance dashboard where you're able to track income and expenses, view budget summary/dashboard, manage savings goals, save their data in browser storage, navigate between multiple sections/pages, etc. 

# Overview
Budget Buddy is a personal finance dashboard designed to help users manage their monthly budget, track expenses, and monitor savings goals in one place.

This application allows users to:
- Enter monthly income
- Track categorized expenses
- Create savings goals with progress bars
- View a financial dashboard summary
- Save all data locally in the browser
